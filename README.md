To test the environment using user input, run main.py
To run the environment using RL agents, run test_agent.py with the appropriate agent (DDPG or DQN)

The settings for the environment can be changed in the test_agent.py file. For the DDPG agent, the action space has to be set to continuous.

Needed packages and modules

 - perlin-noise
 - numpy
 - torch
 - tqdm
 - pygame
 - matplotlib
